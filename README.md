# Nmax

Simple gem for extracting numbers in text and sort them in descendancy order.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nmax'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nmax

## Usage

Nmax::Scan.call(n),
n: total amount of numbers to search
